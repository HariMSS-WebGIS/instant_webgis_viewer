# Changelog — Quick Share Map

## Version 0.1 (2026-06-05)
### Initial Release
- Export any QGIS vector layer to interactive HTML map
- Auto-applies QGIS layer colours and styles
- Statistics panel (feature count, total length/area, CRS, field count)
- Attribute table with pagination (100 features per page)
- Feature search across all attribute values
- Click any feature to see all its attributes in a popup
- 9 basemap options (OSM, USGS Imagery, Topo, Carto Dark/Light, ESRI, etc.)
- Print button for clean map output
- Mobile QR code sharing via web upload
- Works offline — HTML is fully self-contained
- Supports Point, Line, and Polygon layers
- No server, Apache, or configuration required
